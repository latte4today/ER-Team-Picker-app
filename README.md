# ER Team Picker

이터널 리턴 스쿼드 조합 추천 데스크톱 앱입니다.

팀원이 고른 실험체 2명을 입력하면 현재 조합에 어울리는 실험체와 무기 후보를 추천합니다. 내가 실제로 고른 픽은 경기 후 `좋았음` / `별로였음`으로 평가할 수 있고, 누적된 평가는 이후 추천 점수에 반영됩니다.

## 주요 기능

- 팀원 1, 팀원 2, 내 픽을 실험체 카드 클릭으로 빠르게 입력
- 현재 조합에 맞는 추천 후보 표시
- 추천 이유에 역할군, 무기, 평타딜/스킬딜, 교전 기능을 함께 설명
- 사용자 평가 데이터를 기반으로 실험체 간 궁합 점수 반영
- 내가 할 수 있는 실험체 목록을 지정하면 해당 목록 안에서만 추천
- DAK.GG 기반 조합/실험체 랭킹 표시
- 스크린샷 파일, Ctrl+V 붙여넣기, 현재 화면 캡처 지원
- 앱 안에서 문의 보내기 지원
- 다크 모드 UI
- Windows 데스크톱 앱 지원

## 다운로드

GitHub `Releases`에서 최신 설치 파일을 다운로드해 설치할 수 있습니다.

```text
ER-Team-Picker-Setup.exe
```

설치 후 바탕화면 또는 시작 메뉴의 `ER Team Picker` 아이콘으로 실행하면 됩니다.

## 개발 실행

개발용으로 직접 실행하려면 Node.js가 필요합니다.

```powershell
npm install
npm run app
```

PowerShell에서 `npm`이 잡히지 않으면 아래처럼 실행할 수 있습니다.

```powershell
& 'C:\Program Files\nodejs\npm.cmd' run app
```

## 설치 파일 만들기

공유용 앱 폴더를 만들려면:

```powershell
npm run package-win
```

설치 파일을 만들려면 Inno Setup 6 설치 후:

```powershell
npm run installer-win
```

이미 만들어진 `dist` 폴더의 exe는 빌드 당시 코드가 들어간 고정본입니다. 코드를 수정한 뒤 배포하려면 다시 패키징해야 합니다.

## 데이터

추천에는 앱 내부의 실험체/무기 정보, 사용자 평가, DAK.GG 기반 랭킹 데이터가 함께 사용됩니다.

공식 API 연동 전까지는 공개 데이터를 기반으로 메타 데이터를 갱신합니다.

```powershell
npm run collect-dak
```

수집 캐시는 `data/dak-cache/`에 저장되며 GitHub에는 포함하지 않습니다.

## 주의

이 프로젝트는 이터널 리턴 플레이어를 위한 개인 제작 도구입니다. 게임 데이터와 통계는 패치, 메타 변화, 데이터 수집 시점에 따라 달라질 수 있습니다.
